Here are the release notes from v1 to v2.2.0:

v2.2.0 — Full-Featured Local Mode
Local-Only Mode (NEW)
A complete, zero-server period tracker that runs entirely in your browser. Visit /local to use it.

How it works:

You create a passphrase on first use — this generates an AES-256-GCM encryption key via PBKDF2 (210,000 iterations)
All your data (periods, symptoms, journal) is encrypted and stored in your browser's localStorage
The server delivers the page once, then never sees your data — no API calls, no analytics, no cookies
Works fully offline once loaded (installable as a PWA via "Add to Home Screen")
Auto-locks after inactivity (configurable: 2/5/10/30 min)
Export encrypted backups or plain JSON; import from server exports or encrypted backups
Can switch to cloud mode anytime by exporting and importing into a server account
Local mode features (same as server):

Dashboard with cycle phase, predictions, wellness tips, encouragement
Period logging with start/end tracking
Symptom tracker (4 categories, severity, notes)
Journal with mood picker
Calendar with period/predicted/fertile/ovulation markers
Trends: cycle length chart, period duration chart, symptom frequency, symptom×phase correlation, year-at-a-glance mini calendars, month-by-month activity
Settings: cycle/period length, fertility toggle, color theme (Bloom/Ocean), display name, auto-lock timer

TL;DR for users: If you want privacy without trusting any server, use Local-Only Mode at /local. Your data never leaves your device. If you want cross-device sync or partner features, use a server account. You can switch between them anytime via export/import.
